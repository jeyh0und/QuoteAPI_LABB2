Quote API

Do you need a quote to send to your gf or maybe pretend to be a smart and edgy person?

Yes, you've come to the right place.

Use the below mentioned endpoints to navigate the website.

/quotes - Get all the quotes in the database.

/quotes/:id - Find a specific quote by searching for the id.

/addQuote - Fill out a form to add a new quote to the database.

/deleteQuote/:id - Use this link by giving a specific quote ID to remove a quote from the database.
